# Random TODO (macOS)

ランダムにタスクを選び、集中して作業を進めるための macOS アプリです。

## 主な機能
- タスク追加
- ランダム選出
- 作業時間計測（自動）
- 日別履歴保存
- ストリーク（連続達成日数）
- 同じタスクの繰り返し
- 今日の作業時間集計

## 画面構成
- Home: タスク操作・進捗表示
- History: 日別履歴
- Streak: 連続達成状況

## 技術スタック
- SwiftUI
- UserDefaults 永続化
- Combine
